Wiki Editing Tools

I made a few scripts using python to easy the conversion of various json files to the wiki pages. See here: CDDA Wiki Scripts. Some of the scripts might be a bit outdated however, I have not kept up with updating it for 0.E, but am working on it.

Versioning templates

  • The Template:ver, used to create an infobox with the page's version information, and to add a page to that versions category page.
  • The Template:InlineVer, used to create a reference to a specific version without creating the infobox, and tagging the page as being of that version.
  • The Template:mod, used to create an infobox with mod information.

Personal Delete policy

As there are not many people keeping the wiki up to date, I try to nominate as much unused and weird pages as possible. Even redirects that in some little way do make some sense, as to many redirects (for common misspellings for example) make keeping things up to date, and enforcing correct wiki links a pain.
