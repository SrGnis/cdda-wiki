Perhaps we should add '(magiclysm)' to the end of all the links to the monsters and spells. Im worried about overlap between already basegame items/monsters and things added/changed by different mods. --Soyweiser (talk) 00:20, 24 August 2020 (MDT)

I'm not sure how often this would happen with Magiclysm in particular. If there is a possible confusion in a particular case, this would certainly be a reasonable way to distinguish them. -Actual-nh (talk) 23:17, 29 December 2020 (MST)
