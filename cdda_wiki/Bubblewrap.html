“ A sheet of plastic covered with air-filled bubbles. Use it to set it on the ground, creating a trap that will warn you with noise when something steps on it. ”

Notes

Honestly not that useful. The only place where this would be actually useful is outside your door of your bedroom to wake you up if unwanted Intruders arrive. But if you need to set this up, you probably shouldn't be sleeping there anyway. Can often be found in cargo trucks in the back.
