Safemode

From The Cataclysm: Dark Days Ahead Wiki
(Redirected from Auto-Safemode)
Jump to navigation Jump to search

Safemode (previously: run mode) is a measure to prevent dangerous moves. Active Safemode is indicated with SAFE in the lower right corner of the game screen. When Safemode is on and an aggressive monster or NPC is visible, the character will not move without giving a command to ignore the nearby enemy (') or turning safemode off. There will be a message like Spotted jawed terror--safe mode is on! (Press ! to turn it off, press ' to ignore monster) . This is to prevent holding down a key and dying before you've realized that something has been trying to kill you.

The default key to toggle Safemode is !. Safemode can be restored automatically if Auto-Safemode is on and the character does not see a hostile creature for longer than 5 minutes (50 turns). Auto-Safemode can be toggled with the " key.

By default, each game starts with Safemode turned on and Auto-Safemode turned off. This can be changed in the options.
