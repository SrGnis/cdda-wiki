Page merged and about further tweaks

I've merged Whale's wiki page into the existing one. The old entry is now below. Feel free to redistribute the info, delete it or whatever.

The list now contains all the outdated info of the available crafted items. So it needs to be expanded with the new craftable items, and the existing ones should be revised and updated with the info from the latest DDA version.

I've created some redirections to the crafted items, to show how it's done. It's actually pretty easy, if tedious:

  • Click on the link (red color)
  • It'll display an empty page. Click on the Create tab
  • Write there the redirecting code, which works like this: #redirect [[<internal link>]]
  • Submit the page. It'll now display that it's now redirecting to the page you specified. Click there to make sure that the link is OK. Better yet, preview before submitting and try it there.

If you don't know where the correct wiki entry for an item is, just search for it, then link to the appropriate result. More info on how internal links work in MediaWiki here

Just reading the table (could benefit from a few breaks or at least repeating the header info at the bottom; it's kinda long to read in one shot) and it looks like a shelter kit can't be made without having a damaged one. Pretty confident I'm reading it wrong (as otherwise one couldn't make a shelter kit from scratch). Is there a missing OR there? KA101 (talk) 18:02, 13 April 2013 (PDT)

About the shelter, if you pay attention there's two entries. While both are named the same, the first one is the one used to create a shelter from scratch.
