“ This ID card once belonged to a military officer with high-level clearance. It has a magnetic stripe on the back; perhaps it can be used on a control panel. ”

Notes

Military IDs can be used to access missile silos and military bunkers. They are single use as of 0.7 (Lindqvist), so be sure to stockpile a few of them if you want to access more than one military terminal (such as when looting bunkers, which requires at least seven). [Hint]
