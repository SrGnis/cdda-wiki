What version should I download?

Most people play the experimental versions as they run pretty stable and contain a lot of improvements in comparison to the stable versions. If you prefer stability of everything else, then the stable versions are for you. With the experimentals there is always a slight risk of a gamebreaking bug, or a new feature which just isn't working as well as it should, but most people still play the more recent experimentals because they contain more content. You can always ask for a the latest experimental release people are currently using on the C:DDA discord server.

Releases Page

Experimental Builds

These versions are built each time a push is merged with the GitHub repository. They are the latest development builds available and are in no way guaranteed to be stable, or to support your current save game / world (Though many times they are just as stable as the "Stable" versions, if not moreso, and there is some effort to keep saves compatible). Use at your own risk.

Self compile

You can also get the source code from github and then compile it yourself. How_to_compile contains some outdated information on how to compile, which might help.

Various editions of the source code are tagged on github, for example this leads to the 0.7 source code, with some url manipulation you should be able to find other releases.

Changing the font (SDL only)

This will only work with trueType fonts.

You have two options:

  1. Replace the default one, found in data\font, with the one of your choice (simply delete the existing fixedsys.ttf one and copy on that folder the one you want to use).
  2. Edit the path of the FONTDATA file this way:
c:\windows\fonts\< name of your font >
< font width >
< font height >
< font size >

Example: 

c:\windows\fonts\lucon.ttf
8
16
