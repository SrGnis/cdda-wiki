Broad Paws PAWS_LARGE BEAST URSINE
Requires: Paws
Cancels: Large Talons, Paws, Insect Limbs, Insect Arms, Arachnid Limbs, Arachnid Arms, Tentacle Arms, 4 Tentacles, 8 Tentacles
Point cost: -4 / Visibility: 4 / Ugliness: 3
"Your paws are much larger now. Manual dexterity is difficult: permanent hand encumbrance of 20, serious problems crafting, and no gloves. But you can swim more effectively."

(View - Edit Notes )Notes

  • This mutation removes the ability to wear gear in the left hand, right hand location.
  • Adds permanent encumbrance in the following locations: left hand : 20, right hand : 20.
  • Skill modifiers while crafting:
    • electronics gets decreased by: -4.
    • tailor gets decreased by: -4.
    • mechanics gets decreased by: -4.
    • firstaid gets decreased by: -2.
    • computer gets decreased by: -2.
    • traps gets decreased by: -2.
    • fabrication gets decreased by: -2.
    • cooking gets decreased by: -2.
    • survival gets decreased by: -2.
( Edit Notes )
